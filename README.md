# 🎓 Student Registration Portal

A sleek, responsive, multi-step student registration form designed with a modern user interface. This front-end project demonstrates dynamic DOM manipulation, state tracking, and form validation without relying on a back-end database.

## ✨ Features

* **Multi-Step Interface:** Breaks down the registration process into 4 digestible pages:
  1. Personal Information
  2. Contact Information
  3. Academic Information
  4. Account Setup
* **Dynamic Progress Tracking:** Features an interactive UI with numbered step circles and a blue progress bar that updates as the user navigates through the form.
* **Live Avatar Preview:** Includes a custom file upload button that instantly previews the user's uploaded profile picture using the FileReader API.
* **Mandatory Field Validation:** Prevents users from advancing to the next step unless all required fields (`*`) are properly filled out.
* **Modern Dark Theme:** Styled with a clean, dark-mode aesthetic using CSS variables, grid layouts, and smooth transition animations.

## 🛠️ Tech Stack

* **HTML5:** Semantic structure, SVG icons, and input validation attributes.
* **CSS3:** Custom dark mode styling, CSS Grid/Flexbox layouts, and UI keyframe animations.
* **Vanilla JavaScript:** Event handling, DOM manipulation, and paginated form logic .
